# React Design System

A token-based React component library built with TypeScript, TailwindCSS and Storybook.

This project demonstrates how to build scalable, reusable UI foundations that act as a
Single Source of Truth for product teams.

## Goals
- Consistent UI across products and teams
- Strong alignment between Figma and code
- Accessible, responsive and performant components
- Clear contribution and governance principles

## Tech Stack
- React / Next.js compatible
- TypeScript
- TailwindCSS
- Storybook
- Jest / Playwright

## Principles
- Atomic Design
- Token-only styling
- Component-driven development
- Accessibility-first (WCAG)
- Clean and explicit component APIs

## Structure
- Atoms: Buttons, text, icons
- Molecules: Inputs, modals
- Organisms: Headers, tables
- Tokens: Colors, spacing, typography

## Use cases
Designed to be consumed by multiple applications and platforms, ensuring
UI consistency while enabling fast product delivery.
